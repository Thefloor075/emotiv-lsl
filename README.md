
# Emotiv-LSL
=================
## Description
LSL Server for Emotiv EPOC X based on the original code from [CyKit](https://github.com/CyKit/Emotiv-LSL).
### Dependencies
* **Python 3.8** : create a conda environment with Python 3.8
* **Liblsl** : install the liblsl library
* **Pip** : install the necessary packages

## Installation
1. Create a conda environment with Python 3.8
2. Activate the environment
3. Install the dependencies

## Usage
1. **Disable motion data** in Emotiv app settings.
2. **Connect the dongle**, turn on the headset and wait for both indicators to be lit
3. **Run the LSL server**

## Roadmap
* **Windows support** : in development
* **Docker Compose and Docker Run** : in development

## Examples
* **Read raw data** : run a Python script to read the data
* **Write raw data via MNE** : run a Python script to write the data to a .fif file
